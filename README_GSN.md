# RNDR GSN Integration

## Installation

0. Copy `config.example.js` to `config.js` and input required values
1. `yarn`
2. `npm run ganache`
3. `npm run gsn-start`
4. `truffle migrate --reset`
5. `npm start`
6. Go to `localhost:5000`
7. Switch Metamask to local network at `8545` port.
8. Add RNDR token to assets to see your balance on the wallet specified in `config.js`
9. Burn RNDR gaslessly!
